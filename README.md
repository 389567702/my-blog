# 个人技术blog
> 线上地址 [www.binlive.cn](http://www.binlive.cn/).  
> __前台基于  vue+vuex+vue-router+vue-resource+element-ui__  
> 
>__后台用  node(express)+mongodb 搭建__  
>

## clone 项目以后
安装项目依赖
> npm install
> 


运行node服务
> node app.js
> 


然后本地访问
> http://localhost:7000  
> 项目主页
>   

访问后台





> http://localhost:7000/admin  
> 后台管理页面
>
